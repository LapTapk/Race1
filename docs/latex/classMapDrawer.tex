\doxysubsection{Detailed Description}
Компонент, отрисовывающий точки на экране. Способен добавлять точки в \mbox{\hyperlink{classMapCoords}{Map\+Coords}} по нажатию ЛКМ. Сохраняет точки при каждом добавлении.
